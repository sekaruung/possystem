FAQ

What kind of hardware do I need to run the W&O POS?

W&O POS runs on any Android 4.1 or above on phone, tablet, POS Machine or Android emulator.

How to setup printers for W&O POS?

You can choose one of printer connections depends on your store:
1. If you have Windows PC in your store, I will suggest to use W&O POS Printer Adapter. It supports all kind of printers and most stable.
2. If you do not have Windows PC in your store, you can use Wi-Fi/Lan or USB Printers. I will suggest using TSP143 LAN/USB.
3. If you want to use portable printer, I will suggest using Star SM-L200.
If you still do not know how to setup, please reference to Printer Setup Guide or contact us.

How many licenses should I purchase for the local server version?

Each phone/tablet will require each license. For example, if you use two phones, you will require two licenses.

Can I use my license key on another android device?

Each phone/tablet will require each license. For example, if you use two phones, you will require two licenses.

What is different between free version and paid version on Google Play?

Free version means you can purchase features on your need. For example, if you just receipt printer function, just purchase it. Paid version includes all features, excludes other app like KDS, CFD. You can purchase paid version at once for the lower total amount.
